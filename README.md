# 🚀 Sveltoz Website

Modern futuristic drone-inspired website built with React, Vite, and Tailwind CSS.

## ✨ Features

* Responsive Design
* Smooth Animations
* Hero Video Section
* Futuristic UI Effects
* Fast Performance

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS
* JavaScript

## ⚙️ Run Project

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
